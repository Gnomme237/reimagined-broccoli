Site web pour un restaurant

J'ai realisé ce projet pour un client travaillant dans le domaine de le restauration.
L'objectif? Permettre à ce restaurant de gagner en visibilité et de proposer son menu de pizzas aux utilisateurs.

Mon client étant novice dans les nouvelles technologie,ce site donne la possibilité aux utilisateurs de modifier facilement son menu en ligne grace à une interface d'administration sur mesure que j'ai réalisé.

Ce projet simple mais fonctionnel a donner entière satisfaction a mon client.
Technologies utilisées: HTML, CSS, JAVASCRIPT, DJANGO, PYTHON, HEROKU
